PDF to Speech

My this post is a continuation of my previous post regarding speech synthesis from text. You can read it here.

For parsing PDF's I will be using slate and PDFminer. You can install them by running:
sudo pip install --upgrade --ignore-installed slate==0.3 pdfminer==20110515
Here version numbers are important as Slate is built on top of PDFminer and for their proper interaction it is important that they are compatible.

I will be using pyttsx for speech synthesis and espeak to save the spoken text in mp3 format.

First we need to parse the PDF. This is done as 

								loc=raw_input('Enter location of pdf:')
with open(loc) as f:
	doc=slate.PDF(f)
To save the spoken text in mp3, we can follow the following procedure 
								page=input('Enter number of to save in 1 file(0 for all):')
while i< len(doc):
	t=""
	for j in range(i,i+page):
		if i+j>=len(doc):
			break
		t=t+doc[j-1]
	name=(str)(i)+" to "+(str)(i+page)
	i=i+page
	subprocess.call(["espeak", "-w"+name+".mp3", t])
To speak out a specific page we can do 
								page=input("Enter page number to speak:")
t=doc[page-1]
engine=pyttsx.init()
engine.say(doc[page-1])
engine.runAndWait()
