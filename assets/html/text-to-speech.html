Text to Speech

Speech synthesis has always been an interesting field having a wide spread application limited only by one’s imagination. One of the many applications can be an Optical character recognition application that reads out the text from a given photograph. Another from the list can be a pdf reader about which i will be writing in my next post.
Text to speech can be seen in applications like Siri, Cortana, Google Now and all other virtual assistants( I did find Mycroft AI to be really fascinating, and its open source :) )
There are many ways to convert text to speech such as the Google TTS API, one of the best text to speech conversion tool that I tried. But it open only for developers, i.e you need a key from developer console and also Google regularly changes its URL and hence any application built around it often tend to collapse.
Here I will be writing about standalone applications and hence about espeak( Linux) and pyttsx(platform independent).

ESPEAK:

espeak is a cli application for Linux distros that can used to convert text to speech.

Installation:

sudo apt-get install espeak

Usage:

espeak “hello world”
It's as simple as that :)
espeak can also be used to used to save the spoken text in mp3 or other formats.
Try: espeak -h for more options.

PYTTSX:

pyttsx is a python speech synthesizer module. It is platform independent and a lot similar to espeak in terms of accuracy.

Installation:

sudo pip install pyttsx

Usage:

							import pyttsx
content=raw_input(“Enter text to speak:”)
engine = pyttsx.init()
engine.say(content)
engine.runAndWait()
Both espeak and pyttsx are not even close to Google TTS but can be used to make standalone applications.
