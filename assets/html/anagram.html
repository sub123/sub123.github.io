A Game of Anagrams

This article is about a python game which presents players with an anagram and the player has guess the right word within a fixed number of attempts. The number of attempts allowed depends on the length of the word. After each incorrect attempt, user is provided with a hint to the actual word. In case the player is unable to get the right word within the number of allowed attempts, the correct word is displayed and the game continues.

The code discussed here is written on a Linux Mint 17.1 system using Python 2.7 as the scripting language.

In almost all the Linux based systems there is a file located at directory location “/usr/share/dict/” under different names like “cracklib-small”(Ubuntu based systems),”words”(Redhat,Arch) which contains words from dictionary and are often used by many applications to implement features such as “spell-check”.

We just need to parse that file. This can be done as 

								loc=’/usr/share/dict/cracklib-small’
with open(loc) as f:
	content=f.read().split(‘\n’)
f.close()
The file also words like zoo's, 2nd, 3rd which we do not want in our game. Also to avoid making the game too easy I am skipping all the words with length less than 5. All this is done as follows. 
								l=len(content)
words=[]
for i in range(0,l):
	if ‘\” in content[i] or len(content[i])< 5:
		continue
	words.append(content[i])
Finally “words” contain all the words we need for the game to proceed.We can choose a specific word for a particular round of the game by:word=words[random.randint(0,d)] A word would be randomly choosen from the list of words created. To create an anagram of the word we need to shuffle its characters. This is done by: 
								shuffle=list(word)

Now we need to set the number of tries. As this depends on the length of the word it can be done using a simple if-else statement.
During each user attempt we take their input and compare it with the correct word. If they match we congratulate the user and present the next anagram else we provide them with a hint.
To generate hint for a particular word we choose 2 random integers between 0 and length of the word and display the correct characters at those positions. 
								t1=random.randint(0,len(word))
t2=random.randint(0,len(word))
hint=””
for i in range(0,len(word)):
	if i==t1 or i==t2:

	else:
		hint=hint+”.”
