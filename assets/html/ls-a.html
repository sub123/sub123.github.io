ls -a implementation

In bash "ls" command is used to list out the contents of a directory. "ls" with "-a" flag (short for "--all") is used to list out all the files present in a directory including the hidden files.
Here I am going to write about an implementation of "ls -a" command in C/C++.
First we will include all the header files required, "iostream", "dirent.h" and "sys/stat.h". You can read about these header files and different functions implemented in them here

First we need to declare a temporary variable of type dirent which we shall use to loop through all the directory contents. Then we need to declare a variable of data type DIR to store the directory contents. We can dynamically get directory path but here I am defining it. 

								DIR *dir;
struct dirent *file;
dir=opendir("/home/bootstrap");
Name of the file is defined as string "d_name". To get that string we just need to loop through all the files printing their respective "d_name". 
								while(file=readdir(dir))
{
		struct stat st;
		//stat(file->d_name,&st);
		cout<< file->d_name<< endl;
}
closedir(dir);
We can use stat to obtain other informations regarding the file such as permissions, size, timestamp etc. I will be writing about them sometime later. Complie the file using g++ ls-a.cpp -o ls-a.out This will create an executable named ls-a.out. To run executable type ./ls-a.out.
